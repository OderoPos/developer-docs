The Odero Mobile SDK provides a comprehensive solution for integrating payment processing capabilities into mobile applications.

Designed for modern mobile development, it offers seamless integration with the Odero ecosystem while maintaining security and performance standards.

Overview

The Odero Mobile SDK enables developers to:

  • Process card payments securely within mobile applications
  • Process Apple Pay and Google Pay
  • Access Odero's payment infrastructure and services
  • Maintain PCI compliance through secure payment flows
  • Integrate with existing POS systems and merchant accounts
  • Customize the payment experience to match your brand

Key Features

The SDK includes the following capabilities:

  • Cross-platform Support: Compatible with both Android and iOS platforms
  • Secure Transactions: End-to-end encryption and tokenization
  • Easy Integration: Simple APIs and comprehensive documentation
  • Real-time Processing: Fast and reliable payment processing
  • Flexible Configuration: Customizable to meet specific business needs

Supported Platforms

The Odero Mobile SDK supports the following platforms and versions:

  • Android: API Level 26+ (Android 8.0+)
  • iOS: iOS 18.1+
  • React Native: 0.60+

Prerequisites

Before integrating the Odero Mobile SDK, ensure you have:

  • A valid Odero merchant account
  • API credentials (provided by Odero)
  • Development environment set up for your target platform
  • Basic understanding of mobile app development

Getting Started

To begin integration with the Odero Mobile SDK:

  1. Review the platform-specific setup guides
  2. Install the SDK
  3. Configure your API credentials
  4. Implement the payment flow in your application
  5. Test the integration in sandbox environment
