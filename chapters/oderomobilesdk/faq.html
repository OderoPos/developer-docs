Frequently asked questions about the Odero Mobile SDK.

General Questions

Q: What platforms does the SDK support?

A: The Odero Mobile SDK supports iOS (12.0+), Android (API 21+), and React Native (0.60+).

Q: Is the SDK free to use?

A: Yes, the SDK is free to integrate. You only pay transaction fees for processed payments.

Q: Can I use the SDK for testing?

A: Yes, we provide a comprehensive sandbox environment for testing all features.

Integration Questions

Q: How long does integration typically take?

A: Basic integration can be completed in 1-2 hours. Full customization may take longer depending on your requirements.

Q: Do I need to store card information?

A: No, the SDK handles all sensitive card data securely. You never need to store or process card information directly.

Q: Can I customize the payment UI?

A: Yes, the SDK provides extensive customization options for colors, fonts, and layout.

Security Questions

Q: Is the SDK PCI compliant?

A: Yes, the Odero Mobile SDK is fully PCI DSS compliant, ensuring the highest security standards.

Q: How is payment data protected?

A: All payment data is encrypted end-to-end and processed through secure, certified payment processors.

Troubleshooting

Q: What should I do if payments are failing?

A: Check your API credentials, ensure you're using the correct environment, and verify network connectivity. See our Error Handling guide for more details.

Q: How do I contact support?

A: See our Support section for various contact options.
