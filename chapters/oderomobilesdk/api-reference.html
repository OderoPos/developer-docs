The Odero Mobile SDK provides extensive customization options for all UI components. You can change the colors, fonts, and sizes of the user interface elements to match your brand and application design.

UI Customization

Every screen in the payment flow can be fully customized using appearance configuration objects. This allows you to maintain consistent branding throughout the payment experience while ensuring optimal user experience.

Available Appearance Classes

OderoPaymentDetailsPageAppearance

Customize the main payment details screen where users enter their card information, including input fields, labels, buttons, and background styling.

OderoPayWithAnotherCardPageAppearance

Configure the appearance of the screen that allows users to add a new payment method when they have saved cards available.

OderoEnterCVCPageAppearance

Style the CVC entry screen for saved cards, including security code input fields and verification instructions.

OderoPaymentSummaryPageAppearance

Customize the payment summary screen that displays transaction details before final confirmation.

OderoPaymentProcessingPageAppearance

Configure the loading screen appearance while payment transactions are being processed.

OderoPaymentResultSuccessPageAppearance

Style the success screen shown after a successful payment, including success messages, icons, and action buttons.

OderoPaymentResultFailurePageAppearance

Customize the failure screen displayed when a payment fails, including error messages and retry options.
