The Odero Mobile SDK can be integrated into your mobile applications across multiple platforms.

The SDK provides a unified interface for payment processing while handling platform-specific implementations internally.

System Requirements

  • iOS: iOS 18.1+ / Xcode 12.0+
  • Android: API Level 26+ (Android 8.0+) / Android Studio 4.0+
  • React Native: React Native 0.60+

Prerequisites

Before you begin, ensure you have:

  • Valid Odero merchant account
  • API credentials (API Key and Secret)
  • Development environment configured for your target platform

Production Environment - Podfile

# Add to Podfile for production
pod 'OderoPaySDKIOS', :podspec => 'https://raw.githubusercontent.com/OderoPos/react-odero-mobilesdk/main/OderoPaySDKIOS/0.0.1/OderoPaySDKIOS.podspec'

Development Environment - Podfile

# Add to Podfile for development
pod 'OderoPaySDKIOS', :podspec => 'https://raw.githubusercontent.com/OderoPos/react-odero-mobilesdk/main/OderoPaySDKIOS/0.0.1-dev/OderoPaySDKIOS.podspec'

Install dependencies

# Install dependencies
$ cd /path/to/project
$ pod install

Import in Swift files

// Import in Swift files
import OderoPaySDKIOS

iOS Installation

Currently, the Odero Mobile SDK for iOS is only available via CocoaPods installation.

CocoaPods is a dependency manager for iOS projects that simplifies the process of integrating third-party libraries.

The installation process involves:

  1. Your Podfile references the OderoPaySDKIOS.podspec file
  2. The podspec downloads the XCFramework
  3. The XCFramework is hosted in Odero's Artifactory repository

Choose the appropriate version based on your environment:

settings.gradle

// Add the Odero repository to your settings.gradle:
dependencyResolutionManagement {
    repositories {
        maven {
            url = uri("https://ro-artifactory.devtokeninc.com/artifactory/PublicLibraries/")
        }
        google()
        mavenCentral()
    }
}

Production Environment - build.gradle

// Add the library dependency to your app-level build.gradle for production:
dependencies {
    implementation "Odero.oderopaysdk:oderopaysdkandroid:0.0.1"
}

Development Environment - build.gradle

// Add the library dependency to your app-level build.gradle for development:
dependencies {
    implementation "Odero.oderopaysdk:oderopaysdkandroid:0.0.1-dev"
}

Gradle build command

# Build the project with refreshed dependencies:
./gradlew build --refresh-dependencies

AndroidManifest.xml

<!-- Add these permissions to your AndroidManifest.xml: -->
<uses-permission android:name="android.permission.INTERNET" />

proguard-rules.pro

# Add these rules to your proguard-rules.pro:
-keep class com.odero.sdk.** { *; }
-keepclassmembers class com.odero.sdk.** { *; }

Android Installation

The Odero Mobile SDK for Android is available via Gradle dependency management using Odero's Artifactory repository.

Gradle is the build system for Android projects that handles dependency resolution and library integration automatically.

The installation process involves:

  1. Your settings.gradle configures the Artifactory repository
  2. Your build.gradle references the Odero Pay SDK dependency
  3. Gradle downloads the AAR library directly from Odero's Artifactory

React Native Installation

The Odero Mobile SDK for React Native is available via NPM package management.

The installation process involves adding the package URL directly to your package.json dependencies.

Production Environment - package.json

// Add the following dependency to your package.json for production:
{
  "dependencies": {
    "react-native-odero-mobilesdk": "Odero.oderopaysdk:oderopaysdkract:0.0.1"
  }
}

Development Environment - package.json

// Add the following dependency to your package.json for development:
{
  "dependencies": {
    "react-native-odero-mobilesdk": "Odero.oderopaysdk:oderopaysdkract:0.0.1-dev"
  }
}

Install the package using NPM or Yarn:

NPM Installation

# NPM Installation
npm install

Yarn Installation

# Yarn Installation
yarn install

For iOS projects, install the required pods:

iOS Setup

# Navigate to ios folder and install pods:
cd ios && pod install

Android setup is automatic with React Native 0.60+ autolinking.

Import and Usage

// Import and Usage
import { OderoSDK } from '@odero/react-native-sdk';

Note: Make sure to follow the platform-specific installation guide that matches your development environment.
