Testing guidelines and tools for the Odero Mobile SDK integration.

Sandbox Environment

Use the sandbox environment for all development and testing:

OderoSDK.initialize({
  apiKey: 'sandbox_api_key',
  environment: 'sandbox'
});

Unit Testing

// Jest example
describe('Payment Processing', () => {
  test('should process payment successfully', async () => {
    const mockPayment = {
      amount: 1000,
      currency: 'USD',
      reference: 'test_123'
    };
    
    const result = await OderoSDK.processPayment(mockPayment);
    expect(result.status).toBe('success');
  });
});
