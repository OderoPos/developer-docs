Testing guidelines and tools for the Odero Mobile SDK integration.

Sandbox Environment

Use the sandbox environment for all development and testing:

Test Credit Cards

iOS - Apple Pay Testing

For testing Apple Pay integration on iOS, use the official Apple Pay sandbox testing cards. Apple provides comprehensive test card data including different card types, decline scenarios, and authentication flows.

Apple Pay Test Cards: https://developer.apple.com/apple-pay/sandbox-testing/

Android - Google Pay Testing

For testing Google Pay integration on Android, use Google's official test card suite which includes various card networks and payment scenarios.

Google Pay Test Cards: https://developers.google.com/pay/api/android/guides/resources/test-card-suite
