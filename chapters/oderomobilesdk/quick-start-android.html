## 2. iOS Installation ```html:chapters/oderomobilesdk/installation-ios.html

CocoaPods Installation

Add the Odero Mobile SDK to your Podfile:

pod 'OderoMobileSDK', '~> 1.0.0'

Then run:

pod install

Swift Package Manager

Add the following dependency to your Package.swift:

.package(url: "https://github.com/odero/mobile-sdk-ios", from: "1.0.0")

Required Permissions

Add the following permissions to your Info.plist:

<key>NSCameraUsageDescription</key>
<string>This app needs camera access to scan cards</string>
<key>NSLocationWhenInUseUsageDescription</key>
<string>Location is used for fraud prevention</string>

Import the SDK

import OderoMobileSDK
``` ## 3. Android Installation ```html:chapters/oderomobilesdk/installation-android.html

Gradle Setup

Add the Odero repository to your project-level build.gradle:

allprojects {
    repositories {
        maven { url 'https://maven.odero.com/releases' }
    }
}

Add the dependency to your app-level build.gradle:

dependencies {
    implementation 'com.odero:mobile-sdk:1.0.0'
}

Required Permissions

Add these permissions to your AndroidManifest.xml:

<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />

ProGuard Configuration

Add these rules to your proguard-rules.pro:

-keep class com.odero.sdk.** { *; }
-keepclassmembers class com.odero.sdk.** { *; }

Get started with the Odero Mobile SDK in just a few steps. This guide will walk you through the basic setup and your first payment integration.

Quick Start Steps

  1. Initialize the SDK with your credentials
  2. Configure payment settings
  3. Create a payment request
  4. Handle the payment response

Getting Your Credentials

Before you begin, you'll need:

  • API Key: Your unique identifier
  • API Secret: Your authentication secret
  • Environment: sandbox or production

Important: Always use sandbox credentials during development and testing.

``` ## 6. iOS Quick Start ```html:chapters/oderomobilesdk/quick-start-ios.html

1. Initialize the SDK

import OderoMobileSDK

class AppDelegate: UIResponder, UIApplicationDelegate {
    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        OderoSDK.initialize(
            apiKey: "your_api_key",
            environment: .sandbox
        )
        return true
    }
}

2. Create a Payment

let paymentRequest = PaymentRequest(
    amount: 1000, // in cents
    currency: "USD",
    reference: "order_123"
)

OderoSDK.processPayment(request: paymentRequest) { result in
    switch result {
    case .success(let response):
        print("Payment successful: \(response.transactionId)")
    case .failure(let error):
        print("Payment failed: \(error.localizedDescription)")
    }
}
``` ## 7. Android Quick Start ```html:chapters/oderomobilesdk/quick-start-android.html

1. Initialize the SDK

class MainApplication : Application() {
    override fun onCreate() {
        super.onCreate()
        
        OderoSDK.initialize(
            this,
            apiKey = "your_api_key",
            environment = Environment.SANDBOX
        )
    }
}

2. Create a Payment

val paymentRequest = PaymentRequest(
    amount = 1000, // in cents
    currency = "USD",
    reference = "order_123"
)

OderoSDK.processPayment(paymentRequest) { result ->
    when (result) {
        is PaymentResult.Success -> {
            println("Payment successful: ${result.response.transactionId}")
        }
        is PaymentResult.Error -> {
            println("Payment failed: ${result.error.message}")
        }
    }
}
