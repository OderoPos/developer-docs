AndroidManifest.xml

        
        <application>
            <!-- Other fields -->
            <!-- App Name should be started with LYL, thus Launcher can differentiate sale apps -->
            <meta-data
                android:name="app_name"
                android:value="LYL_[APP_NAME]" />

            <meta-data
                android:name="app_model_type"
                android:value="400TR" />

            <meta-data
                android:name="app_version"
                android:value="1" />

            <meta-data
                android:name="sale_activity_name"
                android:value="<internal_packages>.<activity_name>" />

        </application>

In order to be recognized by Token Platform and be visible in the Launcher, please also add the following lines to AndroidManifest.xml file.

        
        <activity
            android:name=".activity.MyActivity"
            android:exported="true"/>

            <!-- Example Usage -->
        <meta-data
            android:name="sale_activity_name"
            android:value=".android.MyActivity" />

Please ensure that sale_activity_name is the Activity that will be shown to the customers as the first screen.
