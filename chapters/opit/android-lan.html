Opit Lan

Dependencies

//in settings.gradle
maven {
    url 'https://ro-artifactory.devtokeninc.com/artifactory/PublicLibraries/'
}

//in app.gradle
dependencies {
   implementation 'OPIT.pos-is-host:common:$commonLibVersion@aar'
   implementation 'OPIT.pos-is-host:clientlib:$clientLibVersion@aar'
   implementation 'OPIT.pos-is-host:common_lan:$commonLibLanVersion@aar'
   implementation 'OPIT.pos-is-host:clientlib_lan:$clientLibLanVersion@aar'
}


You need to add the following dependencies to your build.gradle files (check the right side)

$commonLibVersion = Check latest version here
$clientLibVersion = Check latest version here
$commonLibLanVersion = Check latest version here
$clientLibLanVersion = Check latest version here

After adding the dependencies, you need to sync the project with the gradle files. When the project is synced, you can start using the library.

To use the library, you can follow the instructions from the Bluetooth section.
