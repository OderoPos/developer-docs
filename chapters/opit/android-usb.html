Opit USB

Pos is Host

Dependencies

//in settings.gradle
maven {
    url 'https://ro-artifactory.devtokeninc.com/artifactory/PublicLibraries/'
}

//in app.gradle
dependencies {
   implementation 'OPIT.pos-is-host:common:$commonLibVersion@aar'
   implementation 'OPIT.pos-is-host:clientlib:$clientLibVersion@aar'
   implementation 'OPIT.pos-is-host:clientusb:$clientUsbVersion@aar'
}


You need to add the following dependencies to your build.gradle files (check the right side)

$commonLibVersion = Check latest version here
$clientLibVersion = Check latest version here
$clientUsbVersion = Check latest version here

In this version of the library, the host app is installed on the POS device. The POS device will act as the host and the tablet or other android device will act as the client. The POS device will be connected to the client via USB. The client device will be responsible for sending the commands to the POS device.

If you want to see how the solution works, we can provide you a demo app that you can install on your android client device. For more details please check the Sample Implementation section.

Tablet is Host

Dependencies

//in settings.gradle
maven {
    url 'https://ro-artifactory.devtokeninc.com/artifactory/PublicLibraries/'
}

//in app.gradle dependencies file from the host app (tablet)
dependencies {
   implementation 'OPIT.tablet-is-host:common:$commonLibVersion@aar'
   implementation 'OPIT.tablet-is-host:masterusb:$masterUsbVersion@aar'
   implementation 'OPIT.tablet-is-host:masterlib:$masterLibVersion@aar'
}

//in app.gradle dependencies file from the client app (POS)
dependencies {
   implementation 'OPIT.tablet-is-host:poslib:$posLibVersion@aar'
}


You need to add the following dependencies to your build.gradle files (check the right side)

$commonLibVersion = Check latest version here
$masterUsbVersion = Check latest version here
$masterLibVersion = Check latest version here

These dependencies are for the host app (tablet) using these we will initialize the OPIT library. The host app will be installed on the tablet. The tablet will act as the host and the POS device will act as the client. The tablet will be connected to the POS device via USB. The POS device will be responsible for sending the commands to the tablet.
