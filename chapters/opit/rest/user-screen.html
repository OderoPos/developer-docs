Command: DisplayNumericValueOnScreen

Description: Displays a specified numeric value directly on the user screen of the POS terminal. This command is used to show amounts, quantities, or any other numeric information to the operator or customer as part of the primary display.

The entire request body consists of the following parameters:

  • requestId (string, mandatory): A unique identifier for the request, which can be used for tracking or callback purposes (e.g., "rq_101", "display_total_123").
  • callBack (string, optional): A URL or identifier for a callback function to be invoked after the value is displayed, or if further interaction is expected. Can be an empty string if no callback is needed.
  • text (string, mandatory): The numeric value to be displayed as text on the screen. It is recommended to use a period (.) as the decimal separator for display consistency if the POS system expects this format for non-fiscal display purposes (e.g., "2.25", "100", "0.5").
