App Setup

To integrate your app with the ECR330 app, you need to have the integrator user type. This user type is used to view the apps that are integrated with the ECR330 app.

AndroidManifest.xml

<meta-data
    android:name="app_name"
    android:value="ECR_[app name]" />

If you want to integrate an app with the ECR330 app, you need to mark that app as integrated with the ECR330 app and then the app will be displayed on the main screen of the ECR330. We can do this by adding the app_name metadata tag in the manifest file of the target app. (check the right side)

After that, the app will be displayed on the main screen of the ECR330 app if you are logged in with the integrator user type.


Login in ECR app with integrator user type.

If you don't have this user type, you have to login as admin and create a new user with the integrator type. For this operation you need to have the admin user type and follow these steps:

  1. Login as admin
  2. Go to Programming menu
  3. Go to Users management menu
  4. Click on Add new user button on the top of the screen
  5. Fill in the form with the new user data
  6. Set the user type to integrator
  7. Click on Save button
  8. Logout
  9. Login with the new user
